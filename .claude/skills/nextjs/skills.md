# Next.js Skills

## Next.js Component Creator
**Prompt**: Create a new Next.js 14+ component using TypeScript and Tailwind CSS. The component should follow the App Router convention with server components as default and client components only when necessary using 'use client' directive. Include proper TypeScript interfaces for props and ensure responsive design with Tailwind CSS classes.

## Next.js API Route Generator
**Prompt**: Generate a secure Next.js API route with proper error handling, input validation, and CORS configuration. Use async/await pattern and return appropriate HTTP status codes. Include authentication middleware if required and ensure proper request/response typing.

## Next.js Page Structure Creator
**Prompt**: Create a new page in the Next.js App Router structure with proper metadata, loading states, and error boundaries. Include proper SEO tags, structured data, and follow Next.js best practices for data fetching using fetch or server-side functions.

## Next.js Form Handler
**Prompt**: Implement a form in Next.js with client-side validation, submission handling, and proper state management. Use React Hook Form or similar library for form management and include proper accessibility attributes and error messaging.

## Next.js Authentication Component
**Prompt**: Create an authentication component for Next.js that integrates with common auth providers (Auth.js, Clerk, etc.). Include login, logout, and protected route patterns with proper session management and error handling.

## Next.js Data Fetching Component
**Prompt**: Build a Next.js component that handles server-side data fetching, static generation, or incremental static regeneration based on the use case. Include proper error handling, loading states, and caching strategies.

## Next.js SEO Optimizer
**Prompt**: Enhance a Next.js page or component with proper SEO implementation including meta tags, Open Graph data, Twitter cards, structured data, and performance optimizations following Next.js best practices.

## Next.js Styling Helper
**Prompt**: Apply Tailwind CSS styling to a Next.js component with responsive design, dark mode support, and accessibility considerations. Use Tailwind's utility classes and variant modifiers for different states and screen sizes.
